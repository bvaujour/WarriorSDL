# WarriorSDL

Petit projet personnel pour creer un jeu en C++ en utilisant la SDL2.
Interpretation des fichier JSON generes par le programme TILED pour aider a la creation (tout code a la main).
Programmation orientee objet inspiree de la hierachie de unreal engine (objet, actor, character...).

## Dependencies

```bash
sudo apt install libsdl2-dev libsdl2-image-dev libsdl2-ttf-dev
```

---

## Launching

```bash
make && ./game
```
