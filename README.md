# WarriorSDL

A personal project focused on creating a 2D game in **C++** using **SDL2**.

The project includes a custom parser for **JSON files generated by Tiled**, making level creation easier while keeping all game logic and systems entirely hand-coded. Supported map data includes:

* Player spawn points
* Enemy placement
* Transition zones between maps
* Collision data

The codebase follows an **object-oriented architecture** inspired by the hierarchy used in Unreal Engine, featuring classes such as:

* Object
* Actor
* Character
* And other gameplay-related entities

The goal of this project is to explore game engine architecture, entity management, and level loading while building a complete game framework from scratch.

---

## Dependencies

```bash
sudo apt install libsdl2-dev libsdl2-image-dev libsdl2-ttf-dev
```

---

## Launching

```bash
make && ./game
```
